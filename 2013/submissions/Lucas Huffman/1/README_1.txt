- csc_cont_part1.rb is the submission for Part 1 of the CSC Contest
- testfile.txt is the test file suggested in the problem (other testfiles may be used
but must be placed in the same directory as the *.rb file)
- testfile_counts.txt is the output file containing all of the word counts

- csc_cont_part1_short.rb is the submission for the shortest solution 
Bonus part of Part 1 and requires the same input.It is 325 characters 
long, excluding whitespace and newlines and produces "c.txt" instead
of "filename_counts.txt" like the other version does.
